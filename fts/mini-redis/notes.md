# Mini Redis

- Remote Dictionary Server
- In-memory(hold the data in memory) data structure used as a database, cache, stream engine, and message broker.
- Allow storage, retrieval, and manipulation of various data structures: strings, bitmaps, bitfields, hashes, lists, sets, sorted sets, geospatial, hyperlogs, and streams.
- Key-value store
- 2 primary mechanism for data persistence: `AOF(Append Only File)` and `RDB(Redis Database)`

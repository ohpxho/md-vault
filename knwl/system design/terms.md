# Terms

## What is Fan-out?

- A pattern where one event, message, or data source triggers multiple parallel actions or delivers data to many destinations.
- The rate at which the system can handle multiple outputs

## What is Fan-in?

- A pattern where multiple services/processes send their results or data to a single, central service.
- The rate at which the system can handle multiple inputs

## What is Network Latency?

- The time it takes for a packet to travel from one place to another
- Low latency: fast; High latency: slow
